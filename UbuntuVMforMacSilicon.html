Installing Ubuntu on Mac Silicon

Download the correct iso image.

At the time of this write up, Ubuntu Desktop is not compatible with arm architecture. You have a few options. You can either download a linux iso compatible with arm like Arch Linux or use an x86 Ubuntu iso and attempt to emulate the architecture in your VM (slow and buggy) or you can download the arm based Server Ubuntu OS and just install the gui on top of it. This write up will demonstrate the latter.

First navigate to https://ubuntu.com/download/server/arm and download the iso image. Save the iso to a directory “folder” you plan on keeping all of your VM files in.

Next, you want to download and install the virtual machine. As of writing this document, there are only a few virtual machines that are compatible with Arm and macOS. Navigate to https://mac.getutm.app/ and download and install UTM.

Setting up the VM.

After installing UTM, create a new Virtual machine.

Choose Virtualize.

We will be using the preconfigured settings for Linux, so choose Linux.

For our use case, you can leave Apple Virtualization unchecked. Import the ubuntu ISO you downloaded and continue.

Configure the hardware support. Leave the CPU Cores as default and do not give more than half of macs resources. OpenGL is not needed so leave it unchecked. Continue

Go ahead and assign how much storage to allocate to VM, 64GB should be enough for OS, GUI, and the few applications necessary. Continue.

Leave the shared directory alone and continue.

Give the VM a name and save.

Configure the Operating System.

Now run the Virtual Machine by clicking the play button.

 

After the system boots choose Try or Install Ubuntu Server.

Choose English.

Use the entire disk available to the VM this will populate the 64GB we gave in setup.

Create your login credentials. If forgotten you will need to wipe and start over.

Skip installation of OpenSSH server and just select done.

Skip all of the extra installations and click done.

After installation select reboot. This should restart the VM.

Enter your credentials. Then upgrade all files by typing sudu apt update && apt upgrade 

After the server is done updating, install the gui with command:

sudu apt install ubuntu-desktop

Restart all the default selected serves and press ok.


You will be taken back to the bootup screen. At this point each time you run the Vm select Boot from the next volume.

We are nearly done. Enter credentials and log in. We will install the necessary files missing from the desktop.

Install firefox with command:

sudu apt install firefox

After firefox is installed I recommend running an update through the gui to grab any updates associated with the gui. There will be a few.

At this point you should have all the tools necessary to use Ubuntu Desktop. Any additional programs can be searched and downloaded from firefox. Copy and Paste should also work from your desktop environment to the virtual machine.
